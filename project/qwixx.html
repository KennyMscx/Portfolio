Project

As a team, we programmed the 'Qwixx' board game and developed an online playable version that features multiple rooms, supports an unlimited number of players, and boasts a highly intuitive user interface. This digital adaptation preserves the engaging, fast-paced nature of the original game while enhancing accessibility and user experience. With our seamless design, players can easily navigate the platform, join or create game rooms, and enjoy smooth gameplay with friends or strangers, making it an ideal choice for both casual and competitive play.
